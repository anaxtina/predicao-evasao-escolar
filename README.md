# Projeto: Predição de Evasão Escolar

Este projeto utiliza Inteligência Artificial para prever quais alunos possuem maior risco de abandonar a escola, permitindo uma intervenção pedagógica antecipada.

## Como rodar o projeto
1. Instale as bibliotecas necessárias: `pip install -r requirements.txt`
2. Abra o arquivo `.ipynb` para ver a análise.

## Tecnologias
- Python 3.13
- Scikit-Learn (Random Forest)
- Pandas
